# Documentation Index

This folder contains the standardized documentation set for the Playwright agent initiative.

Terminology used across these docs:
- **Raw evidence**: immutable snapshots, traces, and runtime logs.
- **Compiled memory**: maintained step/locator memory used for cache-first execution.
- **Schema/policy**: rules for confidence, invalidation, contradiction handling, and approvals.
- **Refresh decisions**: `reuse`, `partial_refresh`, `full_refresh`.
